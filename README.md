## Sketch Comedy Database

The code behind [www.SketchTV.lol](https://www.sketchtv.lol). If you find this useful, please give back by adding your favorite sketch!

### Technologies 

- Frontend
  - React
  - MUI
- Backend
  - Next.js 14
  - App Router
  - NextAuth.js
  - ISR
  - AWS S3
- Database
  - Prisma
  - Postgres 
  - SQL Triggers and Functions
- Other
  - TypeScript
  - ESLint
